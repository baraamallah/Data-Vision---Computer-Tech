Component Guide

Disk (Storage)

Permanent storage (HDD/SSD). Holds all programs and files. Data must be moved to RAM before the CPU can use it.

RAM (Memory)

Fast, volatile memory. Stores instructions and data for active programs. Much faster than Disk, but much slower than Cache.

CPU Cache

Ultra-fast memory inside the CPU. It checks for a Hit (data found) or Miss (must go to RAM), significantly speeding up processing.

Decode (Control Unit)

The Control Unit interprets the fetched instruction. It "decodes" what action is needed (e.g., ADD, MOVE, JUMP) before execution.

Execute (ALU)

The Arithmetic Logic Unit (ALU) performs the actual calculation or logical comparison. This is the final step of the Instruction Cycle.
